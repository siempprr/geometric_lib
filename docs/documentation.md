# Документация проекта Geometric Lib

## Общее описание решения
Проект представляет собой библиотеку для вычисления периметра и площади окружности и квадрата. Каждая функция содержит комментарий и пример ее вызова.

## Описание функций

### модуль `circle.py`

#### `area(r)`
Вычисляет площадь круга по радиусу.
**Формула**
S = pi * r * r

**Пример использования**
area(2) -> 12.5663706144

#### `perimeter(r)`
Вычисляет длину окружности по радиусу.
**Формула**
S = pi * 2 * r

**Пример использования**
perimeter(4) ->  25.1327412287

### Модуль `square.py`

#### `area(a)`
Вычисляет площадь квадрата по длине стороны.

**Формула**
S = a * a

**Пример импользования**
area(2) -> 4

#### `perimeter(a)`
Вычисляет периметр квадрата по длине стороны.

**Формула**

P = 4 * a

**Пример использования**

perimeter(4) -> 16
